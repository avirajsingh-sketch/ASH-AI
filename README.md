# ASH-AI
🚀 ASH AI Agent

ASH is not a chatbot.

ASH is a next-generation AI Agent platform designed to think, remember, plan, and assist users intelligently.

Built with a focus on quality over quantity, ASH combines memory, reasoning, task management, user profiles, and intelligent routing into a single experience.

---

✨ Features

🧠 Smart AI Routing

ASH automatically detects user intent and chooses the best reasoning path.

Supported categories:

- Coding
- Research
- Study
- Planning
- Creative Writing
- General Assistance

No manual mode switching required.

---

💾 Persistent Memory

ASH remembers important information and can retrieve it when needed.

- Save memories
- Edit memories
- Delete memories
- Categorized storage
